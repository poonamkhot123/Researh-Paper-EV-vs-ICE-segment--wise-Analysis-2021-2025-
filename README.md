# Researh-Paper-EV-vs-ICE-segment--wise-Analysis-2021-2025-
**EV vs ICE Segment-wise Analysis (2021–2025)** This repository analyzes adoption trends of Electric Vehicles versus Internal Combustion Engine vehicles across 2W, 3W, and 4W segments in India from 2021–2025, focusing on market growth, TCO, policy impact, challenges, and future mobility trends.
